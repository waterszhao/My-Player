# My Player



## description:

Qt 练手的第二个小程序，一个简单的音乐播放器。

可以导入歌曲，管理歌单，设置播放列表等等。

目前只支持本地导入歌曲播放。



可能还存在一些小问题，真的被 Qt 的对象摧毁机制给玩弄了。
